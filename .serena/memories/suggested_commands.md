# Suggested Commands (macOS)
- Install deps: npm install
- Start dev (Metro + platform chooser): npm start
- Run Android device/simulator: npm run android
- Run iOS simulator: npm run ios
- Run web: npm run web
- Export web build: npm run build  (alias build:preview)
- Tests (jest-expo watch): npm test
- Optional: clear Metro cache if stuck: npx expo start -c
